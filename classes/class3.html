Class-3: VARIABLE নামকরণের নিয়মাবলীঃ



VARIABLE নামকরণের নিয়মাবলীঃ বিগত ক্লাসে আমরা variable সম্পর্কে জেনেছি। আজকে আমরা সেই ভেরিয়েবলের নামকরণের নিয়মাবলী অর্থাৎ কিভাবে একটি ভেরিয়েবল ডিক্লেয়ার করতে হয় তা শিখব। এইসব নিয়ম বহির্ভূত কোন ভেরিয়েবল ডিক্লেয়ার করা হলে তা কম্পাইলার গ্রহণ করবেনা অর্থাৎ Error! দেখাবে। তো চলুন দেখে নেয়া যাক কি সেই নিয়মাবলী।

১। Variable নামের শুরুতে নিউমেরিক কী (নাম্বার) ব্যবহার করা যাবেনা। যেমন 1num, 9faysal ইত্যাদি।
২। শুরুতে Alphabetic key (a, b, c etc) ব্যবহার করতে হবে। Lower case (ছোট হাতের) হলে ভালো হয়। যেমন: string, number, sum ইত্যাদি।
৩। শুরুতে Special key (#, %, &, *, $, @, £, ©, œ, π etc) ব্যবহার করা যাবেনা। যেমন: @faysal, #number ইত্যাদি।
৪। শুরুতে এবং ভেরিয়েবলের যেকোন স্থানে Underscore ( _ ) ব্যবহার করা যাবে। যেমন: _string, series_sum ইত্যাদি।
৫। Variable name এর মাঝখানে কোন ফাঁকা যায়গা রাখা যাবেনা। এক্ষেত্রে Underscore দিয়ে সংযুক্ত করতে হবে। যেমন, series_sum, even_number, faysal_ahmed ইত্যাদি।
৬। পাইথনে uppercase এবং lowercase ভিন্ন ভিন্ন ভেরিয়েবল নির্দেশ করে। যেমন Apple & apple দু'টি আলাদা ভেরিয়েবল।
৭। একই ভেরিয়েবল একাধিকবার ব্যবহার না করাই ভালো। একাধিকবার ব্যবহৃত হলে সর্বশেষ ভেরিয়েবলটি গ্রাহ্য হবে। যেমন: 
	a = 10
	a = 20
	print(a)

output :

	20

৮। সর্বোপরী অর্থবহ নাম ব্যবহার করাটাই শ্রেয়। যেমন যোগের জন্য sum, নাম্বারের জন্য number, স্ট্রিং এর জন্য string ইত্যাদি।
