Class-5: If - else statement-


if- else: দৈনন্দিন জীবনে যেমন আমাদের if-else -এর ব্যবহার অত্যাধিক ঠিক তেমনি প্রোগ্রামিংয়েও এর ব্যবহার খুবই গুরুত্বপূর্ণ। আমরা এর ব্যবহার প্রতিনিয়তই করে থাকি। যেমন- আপনি যদি খেলায় অংশগ্রহণ করে প্রথম হোন তাহলে প্রথম পুরস্কার পাবেন। আর যদি দ্বিতীয় হোন তাহলে দ্বিতীয় পুরস্কার পাবেন। আর যদি তৃতীয় হোন তাহলে তৃতীয় পুরস্কার পাবেন। অন্যথায় আপনি পুরস্কার পাবেন না অথবা শান্তনা পুরস্কার পাবেন। এই বাক্যগুলোতে ব্যবহৃত শব্দ 'যদি', 'আর যদি', 'অন্যথায়' এগুলোই মূলত শর্তাধীন শব্দসমূহ।
if else বিবৃতিতে কাজ করার পূর্বে আমাদের সে সম্পর্কে ভালো করে জেনে নিতে হবে।

if, elif, else বলতে যা বোঝায়ঃ if মানে যদি; elif মানেও যদি; তবে একই বিবৃতিতে একাধিকবার if ব্যবহারের প্রয়োজন হলে আমরা শুধু প্রথমবার if ব্যবহার করব, আার বাকিক্ষেত্রে elif ব্যবহার করব। elif কে if এর সমার্থক শব্দও বলা যেতে পারে; else মানে অন্যথায়। if এর সকল স্টেটমেন্ট মিথ্যা হলে else কাজ করবে। if else নিয়ে আরেকটু স্পষ্ট হবে যদি আমরা একটি কোড করে আসি। তো চলুন উপরে বর্ণিত বিবৃতিটিকে প্রোগ্রামিংয়ের ভাষায় রূপান্তর করা যাক,
